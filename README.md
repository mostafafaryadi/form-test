
# Form Test Project - Nextjs


## How to use

[clone the repo](https://github.com/mostafafaryadi/form-test.git):


```sh
git clone https://github.com/mostafafaryadi/form-test.git
```


### Install it:
```sh
npm install
```

### Run In Development Mode:

```sh
npm run dev
```

### Run In Production Mode:

```sh
npm run build
npm run start
```
